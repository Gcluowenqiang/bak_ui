# Changelog

## feat(release): v1.0.0 Initial Release

- Core: 支持增量备份和同步备份两种备份模式
- Core: 基于 ttkbootstrap 的现代化桌面 GUI 界面
- Feat: 支持智能文件比对（文件大小和修改时间），仅复制变更文件
- Feat: 支持同步备份模式，确保目标目录与源目录完全一致
- Feat: 支持实时进度条和日志展示，备份状态实时反馈
- Feat: 支持断点续传，中途停止后下次运行自动跳过已备份文件
- Feat: 支持历史记录管理，自动记录最近使用的源目录和目标目录
- Feat: 支持快速选择历史记录，通过下拉框快速恢复备份配置
- Feat: 集成自动更新检测及更新日志展示
- Feat: 支持安全删除处理，自动处理 Windows 文件权限问题
- Feat: 支持打包为独立 exe 文件，无需 Python 环境即可运行
